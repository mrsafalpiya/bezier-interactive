## Compiling

To compile the program, simply run

```console
$ make release
```

And to install the program system-wide (requires root privilege), run

```console
# make install
```

## Usage

### Mouse

- `Right click` - add/remove control point
- `Left click` - edit a control point (move its location)

### Keyboard

- `i` - invert color
- `t` - toggle info
- `q` - quit
